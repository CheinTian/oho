### Print,printf,println

- [`Printf`](https://golang.org/pkg/fmt/#Printf) - "Print Formatter" this function allows you to format numbers, variables and strings into the first string parameter you give it
- [`Print`](https://golang.org/pkg/fmt/#Print) - "Print" This cannot format anything, it simply takes a string and print it
- [`Println`](https://golang.org/pkg/fmt/#Println) - "Print Line" same thing as `Printf()` however it will append a *newline character*`\n`
